# AI 电商内容生成系统 · SaaS MVP（v0.1）

一个为电商团队量身打造的 AI 内容工作流系统，目标是：
- 面向下沉市场老板，交付“结构化可用”的内容结果
- 支持员工直接用，不依赖复杂 Prompt，操作像“傻瓜相机”
- 初期允许人工参与，逐步替代，最终形成 SaaS 工具化系统
